# CoC Runtime Workflows

## Session

1. doctor
2. campaign list/show
3. read the campaign rule profile
4. retrieve the current scenario scene
5. retrieve recent clues and memory

## Scenario import

1. module inspect
2. apply the `SCENARIO_INDEX.md` quality gates
3. module ingest
4. module index
5. verify scene types, visibility, page ranges, handouts, and solo transitions

## Rule adjudication

1. rules search
2. select the user's imported source
3. rules expand
4. resolve with roll/check/sanity/chase

## Restore

1. save verify
2. explain the target and branch behavior
3. save restore
4. refresh investigator, scene, clues, SAN and memory
