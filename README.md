# SagaSmith CoC Skills

跨平台 CoC 7e Keeper 与战役管理 Skills。

完整模式安装 `sagasmith-core` 和 `sagasmith-coc`。nanobot 与其他平台均通过
Skill 调用 `sagasmith-coc --json`，本仓库不包含数据库、向量、规则引擎或
平台专属 Tools。

商业规则书不随本仓库发布；用户可将自己合法持有的 PDF/Markdown 导入 Runtime。

Skill 覆盖调查员创建与成长、调查和线索、d100/对抗检定、SAN 与疯狂、战斗、追逐、
场景推进、事件/记忆、Snapshot 和恢复。详细流程按需从 `skills/coc7-keeper/references`
加载，主 Skill 保持紧凑。
